About
“This Job Portal is my individual project developed using HTML, CSS, and Bootstrap. The platform is designed to connect job seekers with employers in a simple and effective way. It features separate sections for job seekers and employers, profile management, job postings, and application tracking. The main goal of this project is to practice full-stack concepts while building a real-world inspired web application.”
